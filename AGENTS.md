# AGENTS.md

## 專案目標

建立一個植物大戰殭屍主題的繁體中文遊戲網站，提供遊戲介紹、植物與殭屍圖鑑，以及可逐步擴充的互動內容。

## 技術與結構

- 純 HTML、CSS、JavaScript，不依賴建置工具。
- `index.html`：網站內容與語意結構。
- `styles.css`：視覺與響應式設計。
- `script.js`：互動行為。
- 使用 GitHub Pages 從 `main` 分支根目錄部署。

## 修改原則

- 介面文字使用繁體中文。
- 優先維持手機與桌面都能正常使用。
- 不提交密碼、Token、私密金鑰或個人資料。
- 新增素材前確認授權；避免直接使用未獲授權的遊戲官方素材。
- 修改後至少開啟首頁，確認沒有 JavaScript 錯誤與明顯版面破損。

## Git 工作方式

- 預設分支為 `main`。
- 未經使用者要求，不建立其他遠端服務或付費資源。
- 提交訊息應簡短說明完成的功能。
