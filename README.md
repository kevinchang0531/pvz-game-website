# 植物對戰殭屍：3D 花園塔防

可直接在瀏覽器遊玩的 3D 分路塔防遊戲。玩家收集陽光、種植三種植物，守住五條草坪並擊退五波殭屍。

## 遊戲方式

1. 選擇豆莢砲手、陽光花或岩甲果。
2. 點擊草坪格子種植，陽光花會持續補充資源。
3. 依不同路線配置攻擊與防禦植物。
4. 擋住五波殭屍即可獲勝。

## 技術與素材

- Three.js 3D 場景與互動
- 純 HTML、CSS、JavaScript，可由 GitHub Pages 直接發布
- Kenney Nature Kit 免費 CC0 模型
