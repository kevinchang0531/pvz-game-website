# PVZ 遊戲網站

一個以植物對抗殭屍為靈感的繁體中文靜態網站，目前提供響應式首頁、遊戲特色與角色卡片，後續可擴充圖鑑、攻略與互動小遊戲。

## 本機預覽

直接用瀏覽器開啟 `index.html`，或使用任一靜態網站伺服器。

## 專案結構

- `index.html`：首頁內容
- `styles.css`：網站樣式
- `script.js`：互動功能
- `AGENTS.md`：專案協作規範

## 部署

網站使用 GitHub Pages，從 `main` 分支根目錄發佈。

## 素材說明

目前介面使用文字、CSS 圖形與 Emoji 作為佔位視覺。加入圖片、音效或其他遊戲素材前，請先確認擁有適當授權。
